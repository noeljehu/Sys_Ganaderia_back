# 🐄 Sys Ganadería Backend

Sistema backend para la gestión integral de ganado, enfocado en el control de alimentación, vacunación, peso y ventas de animales. Diseñado para optimizar el seguimiento productivo y sanitario del ganado en entornos rurales o de producción.

---

## 🚀 Descripción del proyecto

Este sistema permite llevar un control centralizado de un hato ganadero, gestionando información clave como:

- Registro y seguimiento de alimentación del ganado
- Control de vacunación y calendario sanitario
- Registro de peso y evolución del animal
- Gestión de ventas de ganado
- Historial individual por animal

El objetivo es mejorar la trazabilidad, productividad y toma de decisiones en la administración ganadera.

---

## 🧠 Características principales

- 📋 Registro de animales
- 🌾 Control de alimentación
- 💉 Gestión de vacunas
- ⚖️ Monitoreo de peso
- 💰 Registro de ventas
- 📊 Historial y seguimiento por animal
- 🔎 Consulta de información detallada
